# Fullscale Test Data

Этот набор используется для полномасштабного тестирования и не должен подменять базовый smoke-набор в `tests/data/*`.

Назначение:
- `positive/` — позитивные сценарии;
- `negative/` — негативные сценарии (валидация, ошибочные форматы, невалидные URL и т.д.).

Важно:
- Валидные изображения/PDF для интеграционных сценариев берутся из существующих `tests/data/images` и `tests/data/pdfs`.
- В этой папке добавлены расширенные кейсы и негативные примеры для регрессии.
